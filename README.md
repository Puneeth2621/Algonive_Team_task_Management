# Team Task Manager

A responsive Team Task Manager web application designed to manage tasks efficiently within a team. 
It supports user authentication, task assignment, status tracking, and real-time task filtering with a clean and modern UI.

---

## Key Features

###  Authentication

 * User Sign Up and Login screens
 * Secure access to the dashboard after authentication

###  Dashboard Overview

* Total Tasks count
* Completed Tasks count
* Overdue Tasks count
* Visual progress indicator for task completion

###  Task Management

* Add new tasks with:

  * Task Title
  * Assigned User
  * Due Date
  * Status (Pending / In Progress / Completed)
* Edit existing tasks
* Delete tasks

###  Filtering & Search

* Filter tasks by status:

  * All
  * Pending
  * In Progress
  * Completed
* Search tasks by assignee name

###  Team-Oriented Design

* Assign tasks to multiple team members
* Track individual responsibilities clearly

###  Data Persistence

* Tasks are stored locally (Local Storage)
* Data remains available after page refresh
